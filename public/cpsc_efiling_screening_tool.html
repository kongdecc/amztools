CPSC 合规与 eFiling 筛查工具

基于 HS 强制申报清单、CPSC/GCC 规则库和 2026 eFiling 生效规则，快速判断证书类型、测试项目、实验室要求和进口申报提示。

CPSC：监管机构

美国消费品安全委员会，负责制定和执行消费品安全法规；常说的“CPSC 认证”通常是指需要提供合规证书。

CPC：儿童产品证书

面向 12 岁及以下儿童设计或营销的产品，如玩具、婴童用品、儿童家具、童装等，通常需第三方检测和 CPC。

GCC：一般合格证书

面向非儿童产品，但受 CPSC 强制规则监管的产品，如部分成人服装、地毯、打火机、家居用品等，通常对应 GCC。

关系可以这样理解：CPSC 管规则，CPC/GCC 是证明产品符合规则的证书；eFiling 则是在进口清关时，把 CPC 或 GCC 的关键证书数据通过 CBP/CPSC 系统电子提交或引用。

HS 强制申报清单查询

官方 HTS 清单用于识别可能被 CPSC/CBP 标记的高风险或可能需证书产品；命中不等于自动需要 eFiling，未命中也不等于免申报。

        eFiling：把 Certificate of Compliance 的关键信息电子化，在进口申报时通过 CPSC PGA Message Set 提交给 CBP/CPSC。它不改变产品本身是否需要证书的判断，但会改变进口时证书数据的提交方式。

        Full PGA Message Set

        报关时提交完整产品证书数据，适合偶尔进口、产品少、同一产品不重复进口的情况。

        Reference PGA Message Set

        先把证书数据录入 Product Registry，再在清关时提交 Certifier ID + Product ID + Version ID，适合重复进口和批量管理。

        Reference PGA 不能只填写进口商 Certifier ID；三项 ID 合在一起才定位到具体产品证书版本。
