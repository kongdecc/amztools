# 头程比价工具

这是一个可直接分享的货代头程报价比价工具。它在浏览器本地解析 Excel，不需要安装、不需要登录，也不需要启动后端服务。

## 怎么用

1. 把整个 `freight-rate-radar` 文件夹发给使用者。
2. 对方打开 `index.html`。
3. 点击“报价文件库”，选择一个或多个 `.xls` / `.xlsx` 货代报价表；文件列表会显示识别到的报价数和仓库数。
4. 输入 FBA 仓库代码，例如 `LAS1`，点击“开始比价”。工具已内置 FBA 仓库库，会提供仓库代码建议并显示仓库地址信息。
5. 可按重量 KG、货代公司、税别、单位、价格上限、时效上限筛选，也可以勾选“每家只看最低”压缩结果。

6. 结果可按价格、时效最快、综合推荐、货代分组或渠道分组查看；也可以复制当前最低价、导出 CSV、导出/导入报价库 JSON。

## 内置 FBA 仓库库

- 当前内置仓库数据共 608 条，来源于你提供的 FBA 仓库数据查询.html。
- 上传报价前也可以看到仓库代码建议；查价时会在提示中显示仓库地区、城市、州省、邮编和地址。
- 上传报价时，如果发现报价里的仓库代码不在内置库或本地新增库中，会自动加入本地仓库库；已存在的仓库会跳过。

## 注意

- Excel 文件只在当前浏览器里解析，解析后的报价索引保存在本机浏览器里，不会上传到服务器。
- 上传后会保存解析好的报价索引，刷新页面后不用重新上传；也可以导出报价库 JSON 给别人导入。
- 请保持 `index.html`、`app.js`、`styles.css`、`xlsx.full.min.js`、`warehouse-data.js` 在同一个文件夹里。
- 浏览器解析版适合随手分享和临时比价；如果要多人共用同一份报价库、自动保存历史文件，才需要再做云端版。
