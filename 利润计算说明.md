
请扮演一位亚马逊资深财务分析师，为我创建一个详细且易用的亚马逊FBA卖家利润计算器。该计算器需要包含所有关键成本项，并内置必要的逻辑判断和说明。
一、计算器基本框架要求：

设计一个输入区（黄色高亮）和一个结果输出区。
输入区：使用预设的固定值或允许用户手动输入。
输出区：自动计算并显示关键财务指标，如回款、毛利、净利润、各项费率等。

二、输入参数与数据来源说明（请在计算器旁添加注释说明）：
A. 核心销售数据

产品售价 (Selling Price): 用户输入。
汇率 (Exchange Rate): 用户输入（用于将人民币成本转换为美元，或反之）。

B. 商品与物流成本
3.  采购成本 (Product Cost): 用户输入（人民币元）。
4.  头程运费 (Shipping Cost to FBA): 用户输入（人民币元/件，指从中国到亚马逊仓库的运费）。
5.  产品重量/尺寸分段 (Weight/Dimensional Tier): 用于自动匹配FBA费用，提供下拉菜单选择（如：小号标准尺寸，大号标准尺寸等）。
C. 亚马逊平台费用 (需根据规则自动计算)
6.  销售佣金 (Referral Fee): 根据售价和产品类目（提供下拉菜单选择，如“服装15%”、“电子15%”、“母婴（售价<$10）8%”）自动计算。公式：售价 * 佣金比例。
7.  FBA配送费 (Fulfillment Fee): 根据B.5中选择的尺寸分段和重量，从内置的费用表中自动匹配数值。（提示：请内置一个根据2025年费率模拟的查询表）。
8.  月度仓储费 (Monthly Storage Fee): 用户可按月输入预估金额，或根据产品体积和亚马逊费率公式估算。
9.  退货处理费 (Return Processing Fee): 根据尺寸分段和重量，从内置费用表中匹配。
D. 营销与推广费用
10. 广告花费 (ACoS): 提供两种输入方式：①输入ACoS百分比，自动计算广告费 = 售价 * ACoS；②直接输入单件产品预估广告费。
11. 促销费用 (Promotion Cost): 如优惠券费用、折扣百分比等，用户输入。
E. 退货与损耗成本 (这是关键难点，需要逻辑说明)
12. 退货率 (Return Rate): 用户输入百分比（数据来源建议：后台“买家之声”或“退货报告”）。
13. 不可售比例 (Unsellable Rate): 用户输入百分比（数据来源说明： 这是退货商品中无法再次销售的比例，需从后台“库存分类账报告”中计算：不可售数量 / 总退货数量。新手可先用30%估算）。
14. 退款管理费 (Refund Administration Fee): 逻辑说明： 亚马逊在处理退款时收取，通常为$5或销售佣金的20%，取较低值。计算器应自动判断并计算。
**退货损失计算公式应为：**
`退货损失 = (售价 + FBA配送费) * 退货率 * 不可售比例 + 退款管理费 * 退货率`

F. 其他杂费
15. 其他费用 (Other Fees): 如VAT/销售税、店铺月租等，用户输入。
三、输出结果要求：

亚马逊回款 (Amazon Payout): = 售价 - 销售佣金 - FBA配送费
毛利润 (Gross Profit): = 回款 - (采购成本 + 头程运费)
总运营成本 (Total Operating Cost): = 广告费 + 促销费 + 仓储费 + 退货损失 + 其他费用
净利润 (Net Profit): = 毛利润 - 总运营成本
净利润率 (Net Profit Margin): = 净利润 / 售价
盈亏平衡ACoS (Break-even ACoS): = 毛利润 / 售价 （这是广告费用的盈亏平衡点）

增加一个“敏感性分析”区域，让用户可以拖动调节“退货率”和“ACoS”两个滑块，实时观察它们对“净利润”的影响。
在关键指标（如净利润）旁添加表情符号（😊/😐😩），直观反映盈利状况。
提供清晰的备注，说明每个数据的来源和估算方法。
